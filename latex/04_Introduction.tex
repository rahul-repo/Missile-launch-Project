Almost all of us have played with rubber bands as it can be used to hit something far away by stretching it to some length and then releasing it. In this project we try to create a mathematical model that relates amount of distance travelled by a rubber band by stretching it some amount. Our model will tell us how much to stretch the rubber band so that it travels a specified length. Initially, We did not have any idea how the model would look like. So as a first step we gathered the data . Then we plotted it . The plot looked like a linear equation. So, we decided to go with linear regression to come up with a mathematical equation. The next sections give details of the project's execution and necessary plots to support our conclusions we have made from the data.\newline
