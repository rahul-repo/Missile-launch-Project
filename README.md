# Missile Launch Project (Rubber Band Projectile Model)

## Overview
This project studies a rubber-band based projectile system framed as a missile launch planning problem. A synthetic rubber band is stretched along a ruler to a known extent and released, and the resulting travel distance is analyzed.

## Objective
To model the relationship between stretch length and travel distance using experimental data and derive a predictive formula with error bounds.

## Methodology
- Data collection from controlled stretching experiments
- Ordinary Least Squares (OLS) regression
- Linear modeling between stretch length and distance traveled
- Error estimation for prediction reliability

## Model
A linear regression model is used:

Distance = f(Stretch Length) + error

## Tools Used
Python, NumPy, Matplotlib, Statistical Regression

## Output
A predictive function that estimates required stretch length for a given target distance.
